🚀 AI Resume Analyzer
An AI-powered Resume Analyzer built using Python, Transformers, Gradio, and Hugging Face Spaces.
This application allows users to upload resumes in PDF format and get:
✅ AI-powered Resume Q&A
✅ ATS Score Analysis
✅ Skill Detection
✅ Resume Summary
✅ Resume Improvement Suggestions

🌐 Live Demo
https://huggingface.co/spaces/santosh70/Resume-analyzer-ai

📸 Features
🤖 AI Resume Question Answering
Ask questions like:
What skills are mentioned?
What projects are listed?
Summarize this resume
What experience does the candidate have?

📊 ATS Score Analyzer
Generates a basic ATS-style score based on resume keywords and structure.

🧠 Skill Detection
Automatically detects technical and professional skills from resumes.
Example:
Python
SQL
Machine Learning
HTML/CSS
Leadership
Communication

💡 Resume Suggestions
Provides improvement suggestions to make resumes stronger and more ATS-friendly.
